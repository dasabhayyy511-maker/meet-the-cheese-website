# Meet The Cheese Website Concept

A responsive single-page restaurant website concept built for Meet The Cheese, covering both the Ajman and Ras Al Khaimah branches.

## Features

- Branded intro logo animation
- Responsive single-page layout
- Signature menu highlights with interactive item details
- Instagram/social highlights section
- Branch contact section for both locations
- Dark premium design inspired by the brand identity

## Project Structure

- `index.html` - main website page
- `styles.css` - visual styling and responsive layout
- `script.js` - intro animation, mobile menu, and menu item interactions
- `Images/` - logo, reel visuals, and supporting brand assets

## How To Run

1. Download or clone the project.
2. Open `index.html` in a browser.

## Deployment

This project can be deployed easily on:

- Netlify
- GitHub Pages
- Vercel

## Notes

This is a website concept/demo created to present the business online in a more polished and professional way.
